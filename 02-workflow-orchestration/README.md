# Workflow Orchestration (Module 2 Zoomcamp)

This folder contains two separate Airflow setups:

1. **Practice DAGs** (`airflow_practice/`)  
   - Dummy DAGs for learning Airflow concepts  
   - BashOperator, PythonOperator, task dependencies  
   - Can run locally via Docker without affecting project DAGs

2. **Project DAGs** (`airflow_project/`)  
   - Actual Zoomcamp workflow orchestration project DAGs  
   - Follows production-style setup  
   - Separate logs, Docker setup, and plugins
